Using a ch unit to size text

the ch unit allows you to control the size od text based on its actual contextual size-the width of a 0 character. this maens that if you set the max width of some text as 70ch: regardles of the size of that text, you can always be sure that it will be at most, 70 characters wide.

Go ahead and change the font sizes and max width of these elements in the CSS editor to see it working
